# NINA - Video Control & New Tab Extension

A powerful Chrome extension that enhances your video streaming experience across multiple platforms and transforms your new tab with a beautiful customizable dashboard.

## Features

### 🎬 Video Control
- **Playback Speed Control**: Hold to fast-forward (customizable 1-10x speed)
- **Auto-Skip Intros**: Automatically skip intros, recaps, and ads on supported platforms
- **YouTube Enhancements**: 
  - Return YouTube Dislikes integration
  - Speed timer display
  - SponsorBlock support
  - Auto-like functionality
- **Prime Video**: Hide X-Ray overlays
- **Multi-Platform Support**: YouTube, Netflix, Prime Video, Crunchyroll, Disney+, and 30+ more platforms

### 📑 New Tab Dashboard
- **Digital Clock**: Real-time display with customizable seconds
- **Quick Shortcuts**: Pre-configured Google services (Mail, Calendar, Drive, Docs, etc.)
- **Custom Shortcuts**: Add your own bookmarks and quick links
- **Smart Search**: Search suggestions from your bookmarks
- **Chrome Bookmarks Integration**: Display and filter your browser bookmarks
- **Appearance Customization**: 
  - Custom background color
  - Background image upload
  - Adjustable overlay darkness

### ⚙️ Settings
- Comprehensive options page
- Multi-language support (English, Deutsch, Español, Français)
- Per-feature toggles
- Persistent settings across sessions

## Installation from Source

1. Clone or download this repository
2. Open `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the NINA folder

## Privacy

We respect your privacy. NINA:
- Never collects personal data
- Works entirely offline (except for external APIs like SponsorBlock, Return YouTube Dislike)
- Stores all settings locally on your device
- Does not require account creation
